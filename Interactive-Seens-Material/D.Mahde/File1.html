Logical Contract

The instruction ADD R2, R1, 15 describes *what* math must happen. How this addition is physically realized via voltage switches, propagation delay paths, or capacitor refreshes is completely abstracted away from the software layer.

Physical Realization Details:

Address 0x100A corresponds to a physical column/row bitline coordinate. The binary state is represented by storing an actual electrical charge in a sub-micron capacitor.

Sub-microscopic physical leaks require memory cells to refresh every ~64ms.

Physical Execution Layer

Instructions translated from memory are decoded into electrical configurations that energize control buses, switch logic gates, routing currents to state registers and charging physical silicon memory slots.
