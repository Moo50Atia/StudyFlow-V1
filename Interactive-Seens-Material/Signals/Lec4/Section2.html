Misconception Buster: "LTI Systems Instant Reaction Myth"

Many students believe LTI systems must stop producing output immediately when the input is removed. This is only true for Memoryless Systems.

For Memoryless:

If input $x(t)$ stops at $t=t_0$, then output $y(t)$ dies instantly at $t=t_0$. The impulse response is confined exactly to $t=0$.

For Systems with Memory:

Systems store historical input energy (e.g. charge inside an RC circuit). The output $y(t)$ continues to persist long after $x(t)$ drops to 0, decay governed by $h(t)$.
