The Decaying Signal Fallacy

Many engineers and students falsely believe that "if an impulse response decays to $0$ as $t \to \infty$, the system is automatically stable."

This is a dangerous mathematical trap! Take the harmonic decay 1 / (t + 1). It gets continuously smaller, yet its accumulation integral grows logarithmically to infinity ($\ln(\infty)$).

Compare this with the stable exponential decay e^{-t} which easily converges to $1$. Absolute integration determines physical stability!

BIBO Stability & Absolute Integrability Guide

A Continuous-Time Linear Time-Invariant (LTI) system with an impulse response $h(t)$ is stable in the **BIBO (Bounded-Input Bounded-Output)** sense if and only if its impulse response is **absolutely integrable**:

\int_{-\infty}^{\infty} |h(t)| dt = S < \infty

What is the Misconception?

Many believe that if $h(t) \to 0$ as $t \to \infty$, the system is BIBO stable. However, decay speed matters. Consider:

  • $h(t) = e^{-at}$ (Exponential Decay): Integrates to $1/a$ (Finite). This system is BIBO Stable.
  • $h(t) = \frac{1}{t+1}$ (Harmonic Decay): The area under this curve behaves like a natural logarithm $\ln(t+1)$. As $t \to \infty$, $\ln(\infty) = \infty$. This system is Unstable!
