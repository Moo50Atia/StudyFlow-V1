Understanding Linearity

For a system T to be **linear**, it must satisfy the principle of superposition:

T{a₁ x₁[n] + a₂ x₂[n]} = a₁ T{x₁[n]} + a₂ T{x₂[n]}
  • Upper Pathway: Sums signals first, then filters them.
  • Lower Pathway: Filters each signal individually, then sums their results.
  • If the resulting waveforms perfectly match, the system is **Linear**.

Understanding Time Invariance

A system is **time-invariant** if a time delay on the input x(t - k) produces an identical time delay on the output y(t - k) without altering shape:

T{x(t - k)} = y(t - k)
  • Upper Path (Delayed Input): Slide slider delay $k$ to shift input first, then pass through system.
  • Lower Path (Delayed Output): Pass input through system first, then shift output by $k$.
  • If the outputs remain identical, the system shape is invariant to shifts!
