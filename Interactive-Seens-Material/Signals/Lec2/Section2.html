Theoretical Insight

A continuous-time sinusoid sin(ω₀t) is always periodic with period T = 2π/ω₀. However, sampling it generates discrete sequences where the index n must be an integer.

For x[n] = sin(Ω₀n) to be periodic, there must exist an integer N > 0 such that: sin(Ω₀(n + N)) = sin(Ω₀n) ⇒ Ω₀N = 2πm ⇒ f₀ = Ω₀/2π = m/N Since m and N must be integers, the frequency ratio f₀ must be a rational fraction!
